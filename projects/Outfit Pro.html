Outfit Pro

Some people spend a considerable amount of time trying on different outfits each morning. To save their time, we developed Outfit Pro, an FPGA-based system that helps visualize various outfit combinations.

  • System Description

System Description

Images of a person in various outfits, taken from a camera connected to an Altera DE2-115 FPGA, are partitioned and saved after performing a background removal algorithm on them.

  • Next

Then, the user previews various outfit combinations by swiping on different sections of a touchscreen, connected to the FPGA, to retrieve the partitioned images saved on the device.

  • Next

All features on the system are implemented using hardware description language.
