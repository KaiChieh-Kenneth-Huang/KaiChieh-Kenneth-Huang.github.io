Taggem

Trying to locate a specific photo on a device can sometimes be difficult and frustrating. This app enables quick photo search by displaying search results in categories according to the tags assigned to each picture.

  • System Description

System Description

Using this application, the user can assign tags to photos on the device and view them in groups according to the tags assigned to them.

  • Next

The file paths of the photos are stored on the app along with the tags assigned. When the user search for photos with a specific tag, the photos are preloaded in cache memory and displayed using the Universal Image Loading Library to reduce loading time.

  • Next

The app also enables users to set alarms that will show the user a randomly selected photo with 3 buttons labeled with a tag name each when they go off. To turn off the alarm, the user must tap on the tag that is associated with the photo displayed.
